HTML5:


Throughout my time at University, I have gained a fair amount of experience with HTML5. C#:


C# is the programming language used primarily through my studies at Deakin University, as such this is my second year using C# as the programming language to undertake most formative assignments and programming challenges. This has helped developed my ability to take a design brief or specification and use C# to develop a solution that addresses this problem. Additionally, within this scope I have undertaken programming challenges using C# as a part of assessment and responsive problem-solving skill development.

PYTHON:


Python has been the most recent programming language I’ve been developing an understanding of. The versatility and popularity of it is the main reason I chose to make this portfolio website using the Python and Django framework. Through this process I have developed a foundational understanding of how python works and it’s ability to be easily compatible and usable in a variety of ways.

DJANGO:


In conjunction with learning python, I have built this web portfolio using the Django framework. This has helped develop my understanding of python as a programming language but also how it can be utilised as a tool to help with overall functionality and usability. This project has been the foundation for developing and programming all back end functionality such as database design, admin functionality and contact page.
